A simple wrapper around the [Skimlinks APIs](http://skimlinks.com/apis)

## Installation

Add this line to your application's Gemfile:

    gem 'skimlinks'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install skimlinks
